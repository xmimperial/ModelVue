
# ModelVue | Professional 3D Model Viewer

ModelVue is a high-performance, web-based 3D model viewer designed for modern engineering and design workflows.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Run development server**:
    ```bash
    npm run dev
    ```
3.  **Explore Documentation**:
    - [Developer Guide](./docs/DEVELOPER_GUIDE.md) - Architecture and coding standards.
    - [Deployment Guide](./docs/DEPLOYMENT.md) - How to ship to production.
    - [Testing Strategy](./docs/TESTING.md) - How we ensure quality.

## Features

- **Multi-format support**: GLB, GLTF, OBJ, FBX, STL, and more.
- **Memory Efficient**: Advanced resource disposal pipeline to prevent GPU leaks.
- **Interactive UI**: Real-time scene controls, wireframe toggles, and metadata analysis.
- **Secure Sandbox**: Client-side parsing ensures your assets never leave your browser unless saved to your profile.
